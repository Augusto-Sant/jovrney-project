How to use REST API:

The api is about journaling so there is a url dedicated to all journals of a user and their pages.

Authentication

To use api its necessary to include token inside headers, to get token send a POST request to /api/auth/ with auth = { 'username':{username}, 'password':{password}, }

Username and password of this website.

Journal

List all Journals

To get a list of all your journals make a GET request to /api/journals/

Get a Journal

To get a journal make a GET request to /api/journals/{journal_id}/

{journal_id} being the primary key of that journal

Update a Journal

To get a journal make a PUT request to /api/journals/{journal_id}/update/

Delete a Journal

To get a journal make a DELETE request to /api/journals/{journal_id}/delete/

Page

List all pages of a Journal

To get a list of all pages of a Journal make a GET request to /api/journals/{journal_id}/pages/

Get a page of a Journal

To get a page of a Journal make a GET request to /api/journals/{journal_id}/pages/{page_id}/

{page_id} being the primary key of that page

Update a page of a Journal

To get a page of a Journal make a UPDATE request to /api/journals/{journal_id}/pages/{page_id}/update/

Delete a page of a Journal

To get a page of a Journal make a DELETE request to /api/journals/{journal_id}/pages/{page_id}/delete/

Search a Word on all pages

Make a get request to Search like /api/journals/{journal_id}/pages/search?q={query}

{query} being the word or phrase parameter to search in pages
